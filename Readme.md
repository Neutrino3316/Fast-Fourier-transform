### 二、实验架构：

- 最低要求：matlab实现二维基-2 库利图基算法、从理论与实操两方面分析算法复杂度、分析内置FTT的复杂度。
- 基础要求：实现课本的逐次加倍法、实现二维基-4 库利图基算法、评估比较四种方法。
- 拔高要求：介绍中国剩余定理、Rader-Brenner算法、Winograd算法

### 三、评估模型：

​	输入为一张图像，输出图像的傅里叶变换矩阵。

​	标准1：衡量计算时间

​	标准2：评估算法与普通DFT的准确度

​	标准3：利用监视窗口查看空间复杂度
